## Verdict

- **CS-001**: raw prints leak control bytes on every attack fixture; the existing line-60 regex strips only colours (still leaks clear/OSC/C1); the allowlist neutralizes all attacks while leaving legit names unchanged.
- **CS-002**: the `get(k) or {}` idiom crashes on nested type confusion, not just top-level scalars; the type-checked helper handles both.
- **CS-003**: clamping percentage alone is insufficient — the poison's `resets_at` dominates the freshness key, so only bounding `resets_at` makes the cache recoverable. `NaN`/`Infinity` (which `json.loads` accepts) are dropped by the `isfinite` check.
